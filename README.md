Soluces for Advent of code 2017 written with TypeScript and mocha tests.

## Build
* clone git
```shell
git clone https://github.com/alocquet/aoc-2017.git
```
* install
```shell
npm i
```
* run all tests
```shell
npm test
```
* run tests only for one day
```shell
npm test -- -g day-05
```
* run tests and watch ts files updates
```shell
npm run watch
```
